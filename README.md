# 🚀 QA Testing Framework

## 📌 Overview
This project demonstrates manual testing, automation testing, and API testing for an e-commerce application.

## 🧪 Manual Testing
- Test Cases
- Bug Reports

## 🤖 Automation Testing
- Tool: Playwright
- Tests:
  - Login
  - Add to Cart

## 🌐 API Testing
- Tool: Axios / Postman
- Validated API response

## ▶️ How to Run

### Automation:
cd automation  
npm install  
npm test  

### API:
cd api-testing  
npm install  
node apiTest.js  

## 💼 Skills Demonstrated
- Manual Testing
- Automation Testing
- API Testing
- Bug Tracking
